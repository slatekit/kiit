Functional 101

An intro to functional programming in Scala.

This is a fairly concise tutorial, with small examples. It does (hopefully) explain that in FP ( functional programming ), you can completely remove the need for some imperative constructus such as variables, return, null, throw.


About

This is a high-level, concept focused introduction to functional programming. While the concepts are language independent, the exampels are in Scala. It covers major paradigms and is intended to provide a guideline to transition to functional coming from imperative languages like C#, Java. Coming soon!
