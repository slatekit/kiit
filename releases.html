Slate Kit

Releases

All the recent and upcoming releases of Slate Kit

Slate Kit is now officially publicly available! Slate Kit has been used in internal projects during the concept and creation phase for 2 years with much success. The investment in Scala, Akka and Akka-Http has paid off tremendously. Scala is a modern, concise, cross-platform language with amazing language features. Akka complements Scala with frameworks to build high-performance, scalable applications. We are open-sourcing a majority ( almost 90% ) of Slate Kit. The Slate.Ext project containing the mobile / web application features are closed-source and will be made available as a premium offering ( coming soon ). The license is an MIT License. Please check our github project for more info.

Get SlateKit from the downloads section below.
