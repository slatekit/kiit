Slate Kit

Releases

All the recent and upcoming releases of Slate Kit

We are very happy to announce that Slate Kit is now officially publicly available. Slate Kit has been used internally on projects for over 1.5 years with much success. The investment in Scala, Akka and Akka-Http has paid off tremendously. Scala is a modern, concise, cross-platform language with amazing language features. Akka complements Scala with frameworks to build high-performance, scalable applications. We are open-sourcing majority ( almost 90% ) of Slate Kit with the exception of the Slate.Ext project containing the mobile / web application features as closed-source and available as a premium offering ( coming soon ). The license is an MIT License. Please check our github project for more info.

Get SlateKit from the downloads section below.

Please report any issues on our github site.
Please feel free to contact us at www.github.com\codehelix or at www.github.com\kishorereddy
