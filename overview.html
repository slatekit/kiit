Overview

A simple Scala toolkit and backend for your mobile and web apps

Overview

  • 100% written in Scala
  • Highly modular utilities and features
  • Layered approach via multiple projects
  • Several utilities available

Philosophy

  • Simplicity above all
  • Modularity in all areas
  • Minimal dependencies
  • Healthy balance between functional vs imperative

1. Slate.Common

Slate Common is the project contain all the core utility code for Slate.

  • 0 dependencies
  • Used by all other projects
  • Contains the core common components

2. Slate.Core

Slate Core contains the components integral to the architecture.

  • Depends on Slate.Common
  • Includes abstractions / base implementations for Queues, Files, Tasks
  • Includes a command line interface component to easily build CLI tools
  • Many other useful components
