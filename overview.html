Overview

A simple Scala toolkit and backend for your mobile and web apps

About

Slate Kit is a utility library, tool kit, and server backend for mobile and web apps. It is written in Scala and uses Akka-Http for the Server. While there are a few server backend projects available either for purchase or open-source, Slate Kit is one of the few written entirely in Scala, Akka-Http and integrated with AWS. Some of the other core differences with Slate Kit is the focus on simplicity, utility code and modularity above all else. One of the other core design philosophies has been to build protocol independent API's that can be "embedded" into a Web API Server or "embedded" into a CLI (Command Line Interface) Shell.

Slate Kit is currently used to build the backend architecture for 2 mobile apps currently in production for 2 client projects of Code Helix Solutions Inc.


Philosophy

Tech 100% Scala for all utilities, features, server code
Simplicity Simplicity above all else
Modularity All utility code and features are modular
Dependencies Kept to a minimum
Configuration Convention over configuration
Balance
  • Mostly immutable + mutable
  • Mostly idiomatic Scala
  • Strongly typed + areas of dynamic programming
  • Functional + Imperative
